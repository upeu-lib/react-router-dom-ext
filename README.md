# Install

```
$ npm i -S react-router-dom-ext

```

# Run demo en modo local

## Clone

```
$ git clone https://github.com/upeu-lib/react-router-dom-ext.git
$ cd react-router-dom-ext

```

## install and start

```
$ npm install
$ npm start

```
